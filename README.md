# Список команд к Git

1) *pwd* - print working directory  
2) *cd* - change directory  
3) *ls* - list directory contents  
4) *touch* - create a file  
5) *mkdir* - create a directory  
6) *cp* - copy  
7) *mv* - move  
8) *cat* - copy contents of a file  
9) *rm* - remove a file  
10) *rmdir* - remove a directory
